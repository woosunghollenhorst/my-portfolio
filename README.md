# Woo-Sung Hollenhorst — E-Portfolio

A personal portfolio website built with [Quarto](https://quarto.org/), managed with Git, and deployed on GitHub Pages.

## Live Site

[woosunghollenhorst.github.io/my-portfolio](https://woosunghollenhorst.github.io/my-portfolio/)

## About

This portfolio showcases my academic and professional projects as part of my MSc in Business Analytics at Dublin City University. It features interactive Python visualisations, an interactive travel map built with D3.js.

## Pages

- **Home** — Landing page with introduction and social links
- **About** — Background, skills, and goals
- **Projects** — Four projects with executable Python code and interactive Plotly charts
- **Reflection** — Insights on skills developed throughout the module
- **Contact** — Contact details and professional links
- **Travels** — Interactive world map of countries visited, personal touch

## Built With

- [Quarto](https://quarto.org/) — Static site generator
- [Python](https://www.python.org/) — Data analysis and visualisation
- [Plotly](https://plotly.com/) — Interactive charts
- [D3.js](https://d3js.org/) — Interactive travel map
- [Bootstrap](https://getbootstrap.com/) — Darkly theme with custom SCSS
- [GitHub Pages](https://pages.github.com/) — Hosting
