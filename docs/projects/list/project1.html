Overview

This project compares two approaches to financial news sentiment classification: fine-tuning a pre-trained BERT model versus using a local Gemma 4 model with zero-shot and few-shot prompting.

Methodology

  • BERT: Fine-tuned bert-base-uncased with Optuna hyperparameter optimisation on a 3-class classification head
  • Gemma 4: Tested with zero-shot and few-shot prompting strategies using the E2B variant
  • Evaluation: Accuracy, precision, recall, and F1-score (macro) across all models

Model Performance Comparison

import pandas as pd
import matplotlib.pyplot as plt

results = pd.DataFrame({
    'Model': ['BERT (Optimised)', 'Gemma 4 (Zero-Shot)', 'Gemma 4 (Few-Shot)'],
    'Accuracy': [0.8000, 0.7480, 0.6834],
    'Precision': [0.8001, 0.7664, 0.7380],
    'Recall': [0.8100, 0.7645, 0.7118],
    'F1 (Macro)': [0.8040, 0.7605, 0.6725]
}).set_index('Model')

fig, ax = plt.subplots(figsize=(10, 5))
results.plot(kind='bar', ax=ax, colormap='Set2')
ax.set_ylabel('Score')
ax.set_ylim(0, 1.05)
ax.set_xticklabels(ax.get_xticklabels(), rotation=15, ha='right')
ax.legend(loc='lower right')
ax.set_title('Model Performance Comparison')
plt.tight_layout()
plt.show()

Key Findings

  • BERT outperformed Gemma 4 across all metrics when fine-tuned on domain-specific data, achieving 80% accuracy vs 74.8% for Gemma zero-shot
  • BERT was ~9x faster for inference — 11.1ms per sample vs 101ms for Gemma 4
  • Gemma 4 zero-shot performed reasonably well without any training, showing strong general language understanding
  • Few-shot prompting did not improve over zero-shot for this task, dropping to 68.3% accuracy
  • Error patterns showed BERT handled nuanced neutral sentiment better, while Gemma was stronger on clearly positive/negative text
